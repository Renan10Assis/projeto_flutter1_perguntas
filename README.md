# projeto_flutter1

Projeto de Flutter de Renan Assis.

## Getting Started

Este projeto é o primeiro projeto da minha trilha de aprendizado de Flutter e Dart, onde é aplicado conceitos da árvore de Widgets seja eles Stateless ou Stateful (com estados), Providers (injeção de dependência reativa e gerenciamento de estado) com Consumers, para que a aplicação renderize/atualize a UI somente do que for necessário (evitando rebuilds desnecessários, otimizando a aplicação).
Nesse projeto, é aplicado no construtor da classe os nominais {super.key, required this.xx} que garante que o Widget seja identificado pelo widget pai e valores dos atributos sejam obrigatórios na instância da classe.
